Overview

BnL Charts provides a D3 charting componentized through Angular directives.

You can mix and match parts and display them using different layout containers. Basic charts require only some HTML and JavaScript configuration. You can also write your own directives and integrate them with existing parts.

Example

This example demonstrates an area chart with x and y axes and a legend.

Layouts

Layout containers allow you to control the placement of the parts within with attributes.

Absolute

Absolute layout supports x, y, width, and height attributes to position parts.

Docked

Docked layout support dock, width, height, and margin attributes.

Nesting

Layouts can be nested to create stacked docked or absolute layout patterns.

Axes

X Axis

The x axis draws the specified scale and support ticks and tick-format attributes.

Y Axis

The y axis draws the specified scale and support ticks and tick-format attributes.

Area

The area plots cartesian data along a filled path.

Legend

The legend provides a color and name reference for multiple series.
